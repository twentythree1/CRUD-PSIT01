PRODUCT HUB

PRODUCT HUB is a functional web application about laptops that's built with the Laravel Framework. This project demonstrates the implementation of the MVC (Model-View-Controller) architecture and provides full CRUD (Create, Read, Update, and Delete) functionality for managing a product inventory.

TECH STACK:
- Framework: Laravel 10.x / 11.x
- Backend: PHP
- Frontend: Bootstrap (UI/UX Design)
- Database: MySQL
- Tooling: XAMPP, Composer, NPM

INSTALLATION & LOCAL SETUP

1. Prerequisites
- Ensure you have XAMPP (with PHP and MySQL) and Node.js installed on your system.
- Composer (Latest Version)
- Laravel (Latest Version)
- VS Code

2. Database Setup
- Open the XAMPP Control Panel.
- Start the Apache and MySQL modules.

3. Start the Application
- You will need two terminal tabs and type these commands:
      - php artisan serve
      - npm run dev
  
The application will be accessible at "http://127.0.0.1:8000/products"

4. Features
- Create: Add new products with names, descriptions, and prices.
- Read: A central dashboard/table view to display all existing records.
- Update: Edit existing product information.
- Delete: Remove products from the system with a confirmation prompt.
